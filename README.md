# Employee Payroll Management System using MySQL
A complete Employee Payroll Management System developed using MySQL to 
demonstrate Database Management System (DBMS) concepts, SQL queries, 
constraints, and payroll calculations.
---
## 📌 Project Overview
This project stores and manages employee information including:
- Employee Details
- Department
- Salary Information
- Tax Details
- Attendance Timing
- Skills
- Employment Status
It also performs payroll calculations such as Gross Salary, Tax Amount, and 
Net Salary.

## Payroll Formula
Gross Salary = Salary + HRA + DA + Bonus
Tax Amount = Gross Salary × Tax Rate / 100
Net Salary = Gross Salary − Tax Amount
